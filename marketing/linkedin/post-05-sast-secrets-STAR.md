**I ran a SAST scan on a fast-moving AI startup.**

47 secrets. 1 codebase. 6 hours before the next prod deploy.

The team was moving at startup speed.

API keys in .env files committed to git.

Tokens hardcoded in test fixtures from 18 months ago.

Credentials buried in a branch nobody had touched in 3 years.

Classic. Fast teams leave tracks.

The task: full secrets audit before the next production deploy.

120 secret patterns. Entropy analysis. Full git history scan.

Done in 15 minutes without touching production.

47 findings. 6 high severity. 3 credentials rotated before exploit.

SARIF output uploaded directly to the GitHub Security tab.

A pre-commit hook was installed the same afternoon.

Zero new secrets committed in the 60 days since.

That is what a single scan buys you.

Not a 6-month security programme. Not a consultancy engagement.

One command. One afternoon. One decision.
